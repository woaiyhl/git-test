### 常规提交
- git add .
- git commit -m 'develop'
- git commit --no-verify -m 'develop'
- git push

### 修改commit的提交信息
- git rebase -i HEAD~1
- 按i键，进入vim模式，将pick替换成r，按esc，输入:wq，再按i键盘，进入vim模式，修改提交信息，再退出vim模式
- 再push

- git commit --amend -m 'new message'
